# Archive Lantern

## Description

An archive lantern is a palm-sized pressure capsule with a cloudy glass core and brass indexing teeth. In low light, it catches two reflections at once: the room around it and the memory stored inside.

## Function

The lantern stores one Afterimage Archive event cluster. A viewer with the right playback rig can experience the recorded fragment, but each playback carries sensory bleed: cold, vertigo, panic, and traces of the original subject's grief.

## History

The Harbor Council adapted old therapeutic offload capsules after memory therapy was banned. The lantern Mara finds under Bellwether Reef contains evidence from the Blackout Night.

## Current State

The lantern is hidden in a flooded archive chamber beneath Bellwether Reef. Venn believes the chamber is still sealed; Mara's salvage work proves otherwise.
