# Worldbuilding

## World Overview

Port Kestrel is a floating harbor built over the drowned remains of an old coastal town. Its residents live with engineered tides, patched storm tech, salvaged pre-flood hardware, and a Council that treats memory as part of civic infrastructure. The sea is not background; it decides work hours, travel routes, debts, and burials.
